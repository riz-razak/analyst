YouTube vs Facebook — what actually pays (SL page, diaspora audience)

Verdict (corrected): Facebook Content Monetization IS live in Sri Lanka (SL added ~Dec 2025) — in-stream ads, ads on Reels, and Stars are available (invite-gated; only the premium Creator Fast Track bonus stays US/CA-only). FB in-stream long-video + diaspora viewers is a real secondary revenue line; YouTube long-form remains the primary anchor (higher RPM, deeper monetization). Use Reels for reach, in-stream for FB revenue; own WhatsApp/newsletter/site.

Switch-on order

1. YouTube YPP first (long-form to diaspora) — the real revenue; enable ads + memberships + Super Thanks.  2. FB Content Monetization (live in SL since Dec '25) — enable in-stream on 3-min+ video (the real FB earner, $1–5/1k) + Reels ads + Stars at ~10k followers / 600k watch-mins; diaspora viewers lift RPM 10–40×.  3. Direct/sponsor on owned surfaces (remittance, travel, telco, money-transfer sponsors) — pays far above ad-share once audience exists.  Build WhatsApp channel + newsletter + own site from day one so you're never one algorithm change from zero.
