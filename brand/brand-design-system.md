# analyst.rizrazak.com — Brand & Design System
**Version 1.0 · March 2026 · No purple**

---

## Philosophy

This is a serious analytical site. The design should feel like a well-funded foreign policy journal crossed with a precision instrument — not a SaaS dashboard, not a portfolio. Colour carries editorial authority. Typography does the heavy lifting. Every element should feel like it belongs in print as much as on screen.

**Three governing principles:**
1. **Restraint is authority** — the less colour you use, the more the colour means
2. **The content is the design** — dossiers, maps, and analysis are the visual material
3. **Forest green replaces purple entirely** — it carries the same calm authority with zero "AI product" association

---

## Palette Options

Five distinct directions. Pick one. Everything else flows from it.

---

### Option 1 — Bawa: Tropical Modernist
*Inspired by Geoffrey Bawa's Heritance Kandalama and Lunuganga — shadow, earth, jungle*

The palette of someone who builds with a place rather than against it. Bawa never used colour as decoration; it came from the materials themselves — teak, terracotta, local plaster. This option treats the interface like an interior: low-contrast, deeply considered, with warm accents that feel earned.

| Role | Name | Hex |
|------|------|-----|
| Primary brand | Jungle | `#1B3A2D` |
| Interactive / links | Canopy | `#3D7A5A` |
| Accent / CTAs | Terracotta | `#A0522D` |
| Background (dark) | Teak | `#2F2B28` |
| Background (light) | Plaster | `#F5F1E6` |
| Supporting warm | Ochre | `#C9A96E` |
| Text primary | Near-black | `#1A1815` |
| Text secondary | Stone | `#7A6F65` |

**Favicon:** Plaster `#F5F1E6` R on Jungle `#1B3A2D` square, radius 18

**Mood:** Aged, material, the opposite of digital-native. Works best if the site leans into long-form dossier aesthetics with generous white space and serif body text.

### Option 2 — Ghibli: Forest Spirit
*Inspired by Princess Mononoke's ancient forest palette — bone, blood, spirit gold against deep green*

Miyazaki's colour philosophy: green is sacred, not decorative. The forest is the protagonist. Every other colour earns its place through contrast with it. This option uses the same logic — deep forest green as the ground, bone white and spirit gold as the light breaking through, a carefully controlled accent red only for urgent states.

### Option 3 — Orange–Teal *(recommended for the admin panel)*
*Complementary tension — deep teal ground, burnt orange action colour*

The most visually dynamic option. Teal and orange are true complementaries — maximum contrast without aggression. This palette has precedent in serious design: mid-century Penguin books, National Geographic maps, brutalist information design. The teal reads as deep water or deep intelligence; the orange as heat, urgency, action.

### Option 4 — Cold War Authority
*Declassified documents, institutional weight, controlled urgency*

The palette of serious power: charcoal + red + slate + gilt. Not frightening, but never casual. This is how a site would look if it were designed by the Government Communications Headquarters and then handed to a good art director. The red is never decorative — it only appears to signal something that matters.

## Typography System

Typography is where editorial authority is actually built. The following system works across all five palette options.

### Option A — Editorial Serif + Geometric Sans *(recommended)*

| Level | Typeface | Weight | Size | Use |
|-------|----------|--------|------|-----|
| Masthead / H1 | **Playfair Display** | 800 | 48–72px | Page titles, dossier heroes |
| H2 | **Playfair Display** | 700 | 32–40px | Section headers |
| H3 | **Inter** | 600 | 20–24px | Sub-sections, card titles |
| Body | **Inter** | 400 | 15–16px / 1.7 leading | Long-form reading |
| Mono / labels | **JetBrains Mono** | 400–500 | 11–13px | Slugs, dates, metadata, code |
| UI / nav | **Inter** | 500 | 13px | Navigation, buttons, badges |

**Loading:** Google Fonts — `Playfair+Display:wght@700;800`, `Inter:wght@400;500;600`, `JetBrains+Mono:wght@400;500`

**Why this works:** Playfair's high-contrast serifs give dossier titles genuine editorial weight. Inter handles everything functional without competing. JetBrains Mono marks metadata and technical detail as distinct from content — readers learn to scan it differently.

## Component States

| State | Treatment |
|-------|-----------|
| Default | Base colour at 100% |
| Hover | `brightness(1.1)` or accent soft |
| Active / pressed | `scale(0.97)` + `brightness(0.9)` |
| Focus | 2px outline in `--color-interactive`, offset 2px |
| Disabled | `opacity: 0.4`, `cursor: not-allowed` |
| Loading | Skeleton shimmer in `--color-bg-mid` |
| Error | `--color-danger` border + icon |
| Success | `--color-success` border + icon |

## Favicon

Custom SVG path R — no font, no circle. Square container with corner radius tuned per palette personality (10–18px). The R uses a geometric construction: vertical stem, closed bowl with curved outer edge, diagonal leg. White or light-tinted R fill against the primary brand dark.

**Files to generate:**
- `favicon.svg` — scalable, used in modern browsers
- `favicon-32.png` — 32×32 ICO fallback
- `favicon-16.png` — 16×16 ICO fallback
- `apple-touch-icon.png` — 180×180 for iOS
